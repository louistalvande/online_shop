== Contexte et objectif

L'Analyse du Besoin Opérationnel (OA) décrit le contexte métier réel d'une boutique en ligne, indépendamment de tout système à construire. Elle identifie les acteurs impliqués, leurs activités opérationnelles et les échanges entre eux.

La boutique repose sur trois acteurs distincts. L'**administrateur** est l'unique responsable de la gestion des comptes utilisateurs. Le **vendeur** (propriétaire du site) prend en charge l'ensemble des opérations commerciales : catalogue, stocks, commandes et expéditions. L'**acheteur** est le client final qui navigue et passe commande.

=== Acteurs opérationnels

[cols="1,1,3", options="header"]
|===
| Acteur | Rôle | Responsabilités principales

| *Administrateur*
| Gestionnaire des comptes
| Créer, suspendre et supprimer les comptes vendeur et acheteurs ; gérer les droits d'accès à la plateforme

| *Vendeur*
| Propriétaire et exploitant de la boutique
| Mettre à jour ses données de compte, gérer le catalogue produits, administrer les stocks, traiter les commandes reçues, gérer les expéditions, consulter les rapports de ventes, traiter les litiges et réclamations

| *Acheteur*
| Client de la boutique
| S'inscrire, mettre à jour ses données de compte, parcourir le catalogue, sélectionner des produits, passer des commandes, effectuer les paiements, suivre et réceptionner les livraisons, laisser des avis
|===

== Diagramme de séquence opérationnel — Passation de commande

Ce diagramme représente les échanges chronologiques entre l'acheteur et l'administrateur-propriétaire lors du processus de commande.

----
skinparam sequenceArrowColor #2E75B6

actor Acheteur as Buyer
participant "Boutique\nen ligne" as Shop
actor Vendeur as Vendor

== Inscription ==
Buyer -> Shop : S'inscrire
Shop -> Admin : Notifier nouvelle inscription
Admin -> Shop : Valider / activer le compte
Shop --> Buyer : Confirmer l'activation du compte

== Navigation et commande ==
Buyer -> Shop : Parcourir le catalogue
Shop --> Buyer : Afficher les produits disponibles

Buyer -> Shop : Ajouter produit au panier
Shop --> Buyer : Mettre à jour le panier

Buyer -> Shop : Valider la commande\n(adresse + paiement)
Shop -> Shop : Vérifier le paiement

alt Paiement accepté
  Shop --> Buyer : Confirmer la commande
  Shop -> Vendor : Notifier nouvelle commande

  Vendor -> Shop : Vérifier le stock
  Vendor -> Shop : Mettre à jour le statut (en préparation)
  Shop --> Buyer : Notifier mise en préparation

  Vendor -> Shop : Déclarer expédition + n° de suivi
  Shop --> Buyer : Notifier expédition avec n° de suivi

  Buyer -> Shop : Confirmer réception
  Shop --> Vendor : Notifier réception confirmée

  Vendor -> Shop : Consulter les rapports de ventes
  Shop --> Vendor : Afficher statistiques et historique

else Paiement refusé
  Shop --> Buyer : Notifier échec du paiement
  note right : L'acheteur peut\nretenter ou annuler
end
----

== Synthèse des activités opérationnelles

=== Administrateur

[cols="1,3", options="header"]
|===
| Activité | Description

| Gérer les comptes utilisateurs
| Créer, suspendre et supprimer les comptes du vendeur et des acheteurs ; gérer les droits d'accès à la plateforme
|===

=== Vendeur

[cols="1,3", options="header"]
|===
| Activité | Description

| Mettre à jour son compte
| Modifier ses informations personnelles (nom, adresse, mot de passe, coordonnées)

| Gérer le catalogue produits
| Créer, modifier et supprimer les fiches produits (photos, descriptions, prix, catégories)

| Gérer les stocks
| Suivre les niveaux de stock, anticiper les ruptures, effectuer les réapprovisionnements

| Traiter les commandes
| Prendre en charge les commandes reçues, vérifier le stock, préparer les colis

| Gérer les expéditions
| Choisir les transporteurs, générer les étiquettes, saisir les numéros de suivi

| Traiter les litiges et réclamations
| Répondre aux demandes de remboursement, gérer les retours produits

| Consulter les rapports de ventes
| Analyser les performances commerciales, le chiffre d'affaires, les produits les plus vendus
|===

=== Acheteur

[cols="1,3", options="header"]
|===
| Activité | Description

| S'inscrire / Se connecter
| Créer un compte (activé par l'administrateur) ou s'authentifier pour accéder aux fonctionnalités d'achat

| Mettre à jour son compte
| Modifier ses informations personnelles (adresse de livraison, mot de passe, coordonnées)

| Parcourir le catalogue
| Rechercher, filtrer et consulter les produits disponibles

| Gérer le panier
| Ajouter, modifier la quantité ou retirer des produits du panier

| Passer une commande
| Saisir l'adresse de livraison, choisir le mode de livraison, valider la commande

| Effectuer un paiement
| Régler la commande par carte bancaire ou autre moyen de paiement disponible

| Suivre la livraison
| Consulter le statut d'expédition et le numéro de suivi du colis
|===
